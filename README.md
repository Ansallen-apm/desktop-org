<a name="english"></a>
## 🇬🇧 English

DesktopOrg is a lightweight, high-performance Windows desktop enhancement tool written in pure C++ and Win32 API. It creates a seamless, transparent organizational layer directly on your desktop, allowing you to categorize files into customizable "Zones" with intelligent auto-sorting.

![DesktopOrg Preview](https://raw.githubusercontent.com/Ansallen-apm/desktop-org/main/assets/preview_placeholder.png) *(Preview image placeholder)*

### ✨ Key Features
- **🎨 Aesthetic Zones**: Create semi-transparent, colored regions on your desktop to group related files and folders.
- **🧠 Intelligent Auto-Sort**: Automatically maps file extensions (e.g., `.pdf`, `.jpg`, `.docx`) to specific zones.
- **🖱️ Zero-Intrusion Interaction**: Drag icons manually into zones to "teach" the system new rules.
- **⌨️ Stealth Mode**: Use `Ctrl + Shift + D` to instantly toggle the visibility of all zones.
- **⚡ High Performance**: Low-level Win32 IPC ensures minimal CPU and memory footprint.
- **🔒 Privacy First**: Runs entirely locally. No telemetry, no cloud.

### 🛠️ Getting Started
1. Download the latest `DesktopOrg.exe` from the [Releases](https://github.com/Ansallen-apm/desktop-org/releases) page.
2. Run `DesktopOrg.exe`.
3. **Controls**: 
   - **Draw**: Click & drag on empty desktop to create a zone.
   - **Modify**: Right-click a zone to change Color, Name, or Rules.
   - **Toggle**: Double-click empty desktop or use `Ctrl + Shift + D`.

## 🇹🇼 繁體中文

DesktopOrg 是一款專為 Windows 設計的輕量、高效能桌面增強工具，採用純 C++ 與 Win32 API 開發。它在您的桌面圖示後方建立一個無縫、透明的組織層，讓您能透過自定義的「區塊 (Zones)」與智慧自動歸類功能，輕鬆整理桌面檔案。

### ✨ 核心功能
- **🎨 美觀區塊**: 在桌面上建立半透明、色彩豐富的區域，將相關檔案與資料夾分門別類。
- **🧠 智慧自動歸類**: 當檔案降落在桌面時，系統會根據副檔名（如 `.pdf`, `.jpg`, `.docx`）自動將其移至指定區塊。
- **🖱️ 無干擾交互**: 手動將圖示拖入區塊，系統會自動「學習」該檔案類型的歸類規則。
- **⌨️ 隱身模式**: 使用 `Ctrl + Shift + D` 即可瞬間切換所有區塊的顯示/隱藏，還您乾淨桌面。
- **⚡ 極致效能**: 底層 Win32 IPC 技術確保極低的 CPU 與記憶體佔用。
- **🔒 隱私至上**: 完全在地端執行，無追蹤、無雲端、無資安疑慮。

### 🛠️ 快速開始
1. 從 [Releases](https://github.com/Ansallen-apm/desktop-org/releases) 頁面下載最新的 `DesktopOrg.exe`。
2. 直接執行 `DesktopOrg.exe` 即可。
3. **操作指南**:
   - **建立區塊**: 在桌面空白處點擊並拖曳即可畫出區塊。
   - **修改設定**: 在區塊上點擊右鍵，可更改顏色、名稱或副檔名規則。
   - **切換顯示**: 在桌面空白處按兩下滑鼠左鍵，或使用快捷鍵 `Ctrl + Shift + D`。
